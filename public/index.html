NotaryPy Badge Generator

This service generates badges for Python package attestations on PyPI.

Usage

Add a badge to your README using this URL format:

![PyPI Attestation](https://notarypy-badge.yourusername.workers.dev/badge/PACKAGE_NAME/VERSION/FILENAME)

API

GET /badge/{package}/{version}/{filename}

This endpoint checks if the specified package has provenance attestations and redirects to a Shields.io badge URL.
